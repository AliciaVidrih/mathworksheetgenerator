# mathworksheetgenerator

Created by Alicia Vidrih

This project is a fully customizable addition and subtraction worksheet generator.
I wrote it in HTML with embedded CSS and JavaScript coding.

Features:
- Addition worksheet generator
- Subtraction worksheet generator
- Customizable header style
- Customizable question count
- Customizable operand ranges
- Customizable sum/difference range
- Customizable sum/difference position
- Customizable missing parameter
- PDF export using jsPDF
